# App

This folder contains the main application code for **AI Notebook Builder**.

## Overview

AI Notebook Builder helps create, edit, and manage educational Jupyter notebooks with AI assistance. The `app` directory holds the core application logic—separate from reference notebooks in `../notebooks/` and one-off utilities in `../scripts/`.
